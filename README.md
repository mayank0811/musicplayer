## Running Locally

To run this project locally -

Clone the project

```bash
  git clone https://github.com/mayank0811/musicplayer.git
```

Go to the project directory

```bash
  cd musicplayer
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
